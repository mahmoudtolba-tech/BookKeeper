# Changelog

All notable changes to BookKeeper Pro will be documented in this file.

## [2.0.0] - 2024-11-09

### Added
- Complete rewrite with modern CustomTkinter UI
- Enhanced database schema with extended book information
- Category system with color coding
- Full lending management system with tracking
- Borrower contact information storage
- Expected return dates and overdue alerts
- Lending history tracking
- Comprehensive statistics dashboard
- Rating system (1-5 stars)
- Publisher, pages, and language fields
- Book descriptions and notes
- Purchase information tracking (price, date, store)
- Export to CSV functionality
- Import from CSV functionality
- Export to JSON functionality
- Database backup and restore
- Dark/Light/System theme switching
- Advanced search with real-time results
- Category filtering
- Modern tabbed interface
- Setup scripts with virtual environment creation (Linux/Mac/Windows)
- Run scripts for easy launching
- Build scripts for creating executables (PyInstaller)
- Comprehensive documentation
- Cross-platform support (Windows, Mac, Linux)

### Changed
- Migrated from basic Tkinter to modern CustomTkinter
- Improved database design with proper foreign keys
- Enhanced search functionality with partial matching
- Better error handling and validation
- Improved code organization with MVC pattern

### Improved
- User interface is now modern, sleek, and intuitive
- Better performance with optimized database queries
- More robust data validation
- Enhanced visual feedback for user actions

## [1.0.0] - Initial Release

### Features
- Basic book management (title, author, year, ISBN)
- Simple Tkinter interface
- SQLite database
- Basic CRUD operations
- Simple search functionality
